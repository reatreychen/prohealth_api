# ProHealth Backend (NestJS)

This is the backend server for the ProHealth Hospital Management System.

## Quick Start
1. Install dependencies: `npm install`
2. Configure `.env` (Database & Cloudinary)
3. Run: `npm run start:dev`

## Key Modules
- **Departments**: Image-enabled hospital section management.
- **Doctors**: Specialty-based doctor profiles.
- **Shop**: Product and category management.
- **Upload**: Dual-mode file handling (Cloudinary/Local).

See the [Master Documentation](../DOCUMENTATION.md) for full details.
